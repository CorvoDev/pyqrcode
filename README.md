# pyqrcode
Create a custom QR Code using pyqrcode

# Requirements
- Python 2.6, Python 2.7, or Python 3

# How to run
```
$ git clone https://github.com/DiabloTheDev/pyqrcode
$ pip install pyqrcode
$ pip install pypng
$ python qrcode.py
```
